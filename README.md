## 🤝 CodeColab - Real-Time Collaborative Coding Platform

🚀 **Excited to introduce CodeColab!** A powerful platform that makes real-time collaborative coding seamless and efficient. Whether you're learning, building a project, or solving problems with a team, CodeColab is designed to enhance your collaboration experience.

## 🛠️ Key Features

* 🔧 **Create a Room**
  Start your collaborative session by generating a unique room.

* 🧑‍🤝‍🧑 **Invite Participants**
  Share the room ID with your peers to invite them to the session.

* 💻 **Real-Time Code Syncing**
  Collaborate on code simultaneously — all changes reflect live for everyone.

* ✨ **Simple and Interactive UI**
  Minimalistic interface for focused collaboration.

## 🧰 Tech Stack

* ⚛️ **React** – Frontend UI
* 🟩 **Node.js** – Backend runtime
* 🚂 **Express.js** – Backend framework
* 🔌 **WebSocket (Socket.IO)** – Real-time communication

## 🚀 How It Works

1. **Visit** the live app link.
2. **Create a Room** — get a unique ID.
3. **Share the Room ID** with others.
4. **Code Together** in real-time!

## ⚙️ Setup Instructions

1. **Clone the repository**

```bash
git clone https://github.com/your-username/codecolab.git
cd codecolab
```

2. **Install dependencies for backend and frontend**

```bash
cd server
npm install
cd ../client
npm install
```

3. **Start the backend server**

```bash
cd ../server
npm start
```

4. **Start the frontend**

```bash
cd ../client
npm start
```

The app should now be running at `http://localhost:3000`
